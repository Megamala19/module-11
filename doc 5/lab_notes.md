# Lab: Clean Up and Push Back to Remote (Git-T03-HOL_002)

Steps performed:
1. Verified `master` was in a clean state (`git status`).
2. Listed all local and remote branches (`git branch -a`).
3. Pulled the latest changes from the remote into `master` (`git pull origin master`).
4. Pushed pending local changes back to the remote (`git push origin master`).
5. Confirmed local and remote `master` were in sync.
